# Lab Notes

The build. Step-by-step records of the zero-trust home lab: network
boundary, DNS, endpoints, virtualization, and the analysis run against
the lab's own telemetry.

Notes are numbered in decade blocks that encode the architecture layer,
not the order they were written:

| Block | Layer |
|-------|-------|
| 00x | Foundation: zero trust framework |
| 01x | Network layer |
| 02x | Endpoints |
| 03x | Virtualization and SOC lab |
| 04x | Analysis artifacts (internal telemetry) |
| 05x | Local AI |
| 06x | Salvage, firmware, and mobile |

The 04x block is the load-bearing distinction: it analyzes this
network's own data (DNS baseline, incident fault isolation). External
landscape analysis lives in threat-intel/ instead.

Committed note numbers are immutable. New notes slot into their block
without renumbering neighbors.
