# CTI Home Lab Portfolio

A running record of a zero-trust home lab build and the analysis it
produces. Three tracks, one repository. This is process documentation:
methodology, tool output, dead ends, and what each step taught. It is
not a polished final product, and that's the point.

## How to read this repo

- **lab-notes/** the build itself, numbered by architecture layer, plus
  analysis of the lab's own internal telemetry.
- **threat-intel/** analysis of the external threat landscape.
- **ctf/** episodic capture-the-flag and Search Party writeups.
- **scripts/** utility scripts referenced by the notes.
- **templates/** the note formats these entries are written against.
- **editing_guide.md** the documentation standard this repo follows.
